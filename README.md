# SYCONNECT
Sistema de Cadastro de Clientes para Escritório de Contabilidade

Esta aplicação foi desenvolvida com o objetivo de centralizar e gerenciar o cadastro de clientes de um escritório de contabilidade, proporcionando mais organização, segurança e agilidade no acesso às informações.

O sistema permite realizar o cadastro completo dos clientes, armazenando os dados em um banco de dados de forma estruturada. Além disso, oferece funcionalidades para consultar, editar e atualizar informações sempre que necessário, garantindo que os registros permaneçam corretos e atualizados.

O projeto foi pensado para facilitar a rotina do escritório, reduzindo o uso de planilhas e processos manuais, além de contribuir para um gerenciamento mais eficiente das informações dos clientes.

Principais funcionalidades
Cadastro de novos clientes;
Armazenamento dos dados em banco de dados;
Consulta de clientes cadastrados;
Edição e atualização de informações;
Organização e gerenciamento dos registros;
Interface simples e intuitiva para facilitar o uso.

Este projeto faz parte do meu processo de aprendizado e aprimoramento como desenvolvedor, aplicando conceitos de desenvolvimento web, integração com banco de dados e operações de CRUD (Create, Read, Update e Delete).
